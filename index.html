🖧 개념 2 · 정보 통신 기술의 구성

정보 통신 기술은 크게 정보 처리 시스템(컴퓨터 내부)과 정보 전송 시스템(통신 장치)으로 구성됩니다.

💻

정보 처리 시스템

컴퓨터 본체 안에 있는 장치들
정보를 수집·저장·처리·출력합니다
🖥️ 컴퓨터 = 정보 처리 시스템의 핵심
🧠
CPU (중앙 처리 장치)
제어 기능 + 연산 기능 — 컴퓨터의 두뇌
예: Intel Core i9, AMD Ryzen, Apple M3
💾
주기억 장치 (RAM / ROM)
현재 실행 중인 프로그램을 임시 저장 (전원 끄면 사라짐)
예: DDR5 RAM 16GB, 바이오스 ROM
🗄️
보조 기억 장치
파일·운영체제를 영구 저장 (전원 꺼도 유지)
예: HDD 1TB, NVMe SSD, USB, DVD
⌨️
입력 장치
아날로그 신호를 디지털로 변환해 CPU로 전달
예: 키보드, 마우스, 마이크, 터치스크린
🖥️
출력 장치
디지털 처리 결과를 사람이 알 수 있는 형태로 출력
예: 모니터, 프린터, 스피커
🔌
메인 보드 (마더보드)
모든 부품을 연결·데이터 교환·전원 분배
예: ASUS ROG, MSI MAG, Gigabyte Z790
📡

정보 전송 시스템

컴퓨터와 컴퓨터를 연결하는 통신 장치들
정보를 한 곳에서 다른 곳으로 전송합니다
🌐 인터넷 = 정보 전송 시스템의 연결망
💻
단말 장치
정보를 수집하고 전송 가능한 형태로 변환
예: 키보드, 디지털 카메라, 마이크, 캠코더
📠
신호 변환 장치 (모뎀)
아날로그 ↔ 디지털 신호 양방향 변환
예: DSL 모뎀, 케이블 모뎀, ADSL
🌐
전송 회선
전기·광 신호가 이동하는 물리적 통로
예: 광 케이블, 동축 케이블, Wi-Fi, 5G
🖧
통신 제어 장치 (라우터)
올바른 목적지까지 데이터 경로를 안내
예: Wi-Fi 공유기, 스위치, 게이트웨이
📌 핵심 정리: 카카오톡 메시지를 보낼 때 → 정보 처리 시스템(내 컴퓨터/폰)에서 데이터가 만들어지고, 정보 전송 시스템(모뎀·회선·라우터)을 거쳐, 다시 정보 처리 시스템(친구 폰)에서 화면에 표시됩니다!

🖥️ 개념 3 · 컴퓨터의 6가지 구성 요소

컴퓨터(정보 처리 시스템)는 아래 6가지 핵심 장치로 구성됩니다. 각 카드를 읽고 역할을 기억해 보세요!

CPU
중앙 처리 장치
🧠 컴퓨터의 두뇌!
모든 계산(연산)과 명령(제어)을 처리하는 핵심 부품입니다.
🔧 제어 기능 — 다른 장치에 "이렇게 해라!" 명령
🔢 연산 기능 — 더하기·비교 등 계산 처리
클록 속도(GHz)가 높을수록 더 빠르게 처리해요.
Intel Core i9 AMD Ryzen Apple M3
RAM
주기억 장치 (Random Access Memory)
📋 작업용 책상 위 공간
현재 실행 중인 프로그램을 임시로 올려두는 작업 공간입니다.
📌 전원을 끄면 내용이 모두 사라져요 (휘발성)
📌 ROM은 부팅 프로그램을 영구 저장하는 읽기 전용 메모리
용량이 클수록 더 많은 프로그램을 동시에 실행 가능해요!
8GB / 16GB / 32GB ROM (바이오스)
메인 보드
마더보드 (Main Board)
🏗️ 건물의 뼈대 + 복도
모든 부품을 연결하고 데이터를 주고받게 해주는 기반 보드입니다.
📌 CPU·RAM·SSD·그래픽카드 등을 슬롯에 장착
📌 부품 간 데이터 이동 경로 버스(고속도로) 제공
📌 BIOS/UEFI로 컴퓨터 기본 설정 관리
ASUS ROG MSI MAG ATX / mATX
보조 기억 장치
HDD / SSD
🗂️ 집의 책장·서랍장
전원을 꺼도 데이터가 영구적으로 저장되는 비휘발성 장치입니다.
📌 HDD: 자기 원판에 저장, 가격 저렴
📌 SSD: 플래시 메모리, 속도 훨씬 빠름
운영체제(Windows), 사진, 게임, 문서 파일이 모두 여기에!
HDD 1~4TB NVMe SSD USB / DVD
입력 장치
Input Device
🎤 컴퓨터에게 말 거는 입
사람이 컴퓨터에게 정보를 전달하는 장치입니다.
📌 아날로그 신호를 디지털 신호로 변환해 CPU로 전달
📌 키보드: 문자, 마우스: 위치, 마이크: 소리 입력
터치스크린은 입력·출력 동시 가능한 특수 장치!
키보드 마우스 터치스크린 마이크 스캐너
출력 장치
Output Device
📺 컴퓨터의 목소리·얼굴
컴퓨터가 처리한 결과를 사람이 알아볼 수 있게 표시하는 장치입니다.
📌 디지털 데이터를 → 빛·소리·종이로 변환
📌 모니터: 시각, 스피커: 청각, 프린터: 인쇄
VR 헤드셋, 홀로그램도 출력 장치의 미래 형태예요!
