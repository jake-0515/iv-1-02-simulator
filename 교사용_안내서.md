## 🚀 실행 방법

**방법 1: 더블클릭 (가장 간단)**
- `index.html`을 더블클릭하면 기본 브라우저(크롬 권장)에서 바로 실행됩니다.

**방법 2: 수업용 스크린**
- 교사용 컴퓨터에서 `index.html`을 열고, 학생들이 각자 기기에서도 동일 파일을 열도록 안내.
- 인터넷 연결 없이도 동작합니다 (완전 오프라인).

**방법 3: 간이 웹 서버 (네트워크 공유 시)**
```
cd "D:\2026학년도\2026 기술수업\IV-1-02_시뮬레이터"
python -m http.server 8080
```
→ 같은 와이파이에 있는 학생 기기에서 `http://[교사IP]:8080` 접속.

## 💡 수업 팁

1. **시뮬레이터 2의 함정:** 부품 보관함에 모뎀과 라우터가 섞여 있습니다. 학생들이 "왜 이게 본체에 안 들어가?"라고 묻는 순간을 노려 정보 처리 시스템 vs 정보 전송 시스템의 구분을 자연스럽게 지도하세요.

2. **신호 변환 활동:** 샘플링 슬라이더를 4 → 32로 이동시키면 디지털 신호가 점점 원본 아날로그에 가까워집니다. "CD 음질 vs MP3 음질" 비교로 연결하면 학생들이 즉시 이해합니다.

3. **모둠 활동:** 4명 모둠에서 "장치 분류 챌린지"를 릴레이로 진행 → 한 명씩 번갈아 분류. 협동 + 경쟁 학습.

4. **크롬 권장:** 드래그 앤 드롭과 Canvas 애니메이션이 가장 부드럽게 동작합니다. 사파리/엣지도 정상 작동합니다.

## 📞 문제 해결

| 증상 | 해결 |
|---|---|
| 글자가 깨져 보임 | UTF-8 한글 폰트가 없는 브라우저. 크롬 최신판 사용 권장. |
| 드래그가 안 됨 | 터치 기기에서는 부품을 **탭** → 슬롯/카테고리 **탭** 방식 사용 |
| 이모지가 안 보임 | Windows 7 이하 일부 기기. Windows 10 이상 권장 |
| 이미지가 안 나옴 | 본 자료는 이미지 파일을 쓰지 않고 이모지/SVG만 사용. 문제 없음. |
